PayPal payment buttons

Integrating with the experimental PayPal JavaScript buttons is as easy as including a small snippet of code. We have multiple flavors of buttons to fit your needs.

Pick a button, click Customize to make it your own, and try the generated button to make a PayPal payment.

Important: Payments made using this code will be live and not Sandbox test payments.

  • Buy Now
  • Add to Cart
  • Hosted
  • QR Codes

Buy Now buttons are perfect for single item purchases Customize

<script src="paypal-button.min.js?merchant="
    data-button="buynow"
    data-name="My product"
    data-amount="1.00"
></script>
Email address or merchant ID

Add To Cart buttons lets users add multiple items instantly giving your website a cart Customize

<script src="paypal-button-minicart.min.js?merchant="
    data-button="cart"
    data-name="Product in your cart"
    data-amount="1.00"
></script>

Add payment buttons to your site

PayPal makes it easy to accept payments on a website, whether it's a mobile device, tablet, or PC. Just copy and paste a short HTML snippet on your website to add a payment button or a shopping cart.

  1. Get your Merchant account ID or email address. You'll need to add it to the code above.

    If you're logged in to PayPal, you can get find it on your account profile page.

  2. Download one of these two JavaScript files and add it to your site, depending on whether you want a simple button or a cart. You need to link to it in the code above:

    • JavaScript Buttons
      Basic support for PayPal's JavaScript buttons

    • JavaScript Buttons + MiniCart
      Supports PayPal's JavaScript buttons, plus enhances your cart buttons with the PayPal Mini Cart.

  3. Customize the button

    Choose your button type, and customize your button with your product name and price using the Customize option.

  4. Add the button code to your website.

    Just copy and paste the HTML snippet, and you're ready to start selling!

    Important: Payments made using this code will be live and not Sandbox test payments.

It's that simple. No need to deal with a long confusing form. You can update product name and pricing directly on your site. These JavaScript buttons are easy, fast, and secure.

If you are familiar with the classic PayPal button solution, it is still available.

Browser support

The JavaScript buttons have been tested and work in all modern browsers including: Chrome, Safari, Firefox and Internet Explorer 7+.
