PayPal payment buttons

Integrating with the experimental PayPal JavaScript buttons is as easy as including a small snippet of code. To get started:

  1. Customize your button
  2. Add the button to your site

Customize your button

  1. Click Customize to customize your button with your merchant ID, product name, price and other options.
  2. Click on the generated button to test your button.
  • Buy Now
  • Add to Cart
  • Donate
  • QR Codes

Buy Now buttons are perfect for single item purchases Customize

<script src="paypal-button.min.js?merchant=YOUR_MERCHANT_ID"
    data-button="buynow"
    data-name="My product"
    data-amount="1.00"
></script>
Email address or merchant ID

Add To Cart buttons let users add multiple items instantly giving your website a cart Customize

<script src="paypal-button-minicart.min.js?merchant=YOUR_MERCHANT_ID"
    data-button="cart"
    data-name="Product in your cart"
    data-amount="1.00"
></script>
Email address or merchant ID

Donate buttons are great for accepting donations from users Customize

<script src="paypal-button.min.js?merchant=YOUR_MERCHANT_ID"
    data-button="donate"
    data-name="My product"
    data-amount="1.00"
></script>
Email address or merchant ID

QR codes are ideal for allowing quick mobile payments Customize

<script src="paypal-button.min.js?merchant=YOUR_MERCHANT_ID"
    data-button="qr"
    data-name="Product via QR code"
    data-amount="1.00"
    data-size="250"
></script>
Email address or merchant ID

Important: Payments made using this code will be live and not test payments.

Add the button to your site

  1. Get your Merchant ID or email address. You'll need to add it to the code above. If you're logged in to PayPal and have a business account, you can get your Merchant ID on your account profile page.

  2. Download one of these two JavaScript files and add it to your site, depending on whether you want a simple button or a cart.

    • JavaScript Buttons
      Basic support for PayPal's JavaScript buttons

    • JavaScript Buttons + MiniCart
      Supports PayPal's JavaScript buttons, plus enhances your cart buttons with the PayPal Mini Cart.

    In the generated button code, you can edit the path to the PayPal Button JavaScript file, if needed.

  3. Copy and paste the HTML snippet you customized above, and you're ready to start selling!

Customization

For advanced needs, there's a JavaScript API that can be used to pragmatically create buttons, localization support, and the ability to create user-editable inputs. Find out how in the README file.

Browser support

The JavaScript buttons have been tested and work in all modern browsers including: Chrome, Safari, Firefox and Internet Explorer 7+.
